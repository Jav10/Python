OBJETO

Un objeto es cualquier cosa.

Cosa Definición: Lo que tiene entidad, ya sea corporal o espiritual, natural o artificial, concreta, abstracta o virtual. Objeto inanimado, por oposición a ser viviente.

Objeto: instancia de una clase.

Un objeto tiene características, propiedades que pueden describirlos (atributos), y también tienen comportamientos (métodos) y son distintos para cada objeto, cada uno tiene su propia identidad, podríamos tener dos autos son dos objetos, pero cada uno tiene su propia identidad, ya que uno puedo ser blanco y el otro negro, uno de una marca y el otro de otra, uno de cuatro cilindros y el otro de dos.

Identidad, atributos y comportamientos.

El o La para identificar objetos, sustantivos -> Objetos - Verbos -> Acciones.

CLASE

Clase Definición: Conjunto de elementos con caracteres comunes.

Podríamos decir que una clase es como algún tipo de plano (clase), supongamos un plano para crear un avión, con ese plano podremos describir cómo será el avión y podremos construir el avión (objeto) a partir de este plano y no sólo uno sino muchos objetos. Entonces para construir objetos primero debemos crear la clase.

Ideas fundamentales de POO

  • Abstracción Definición: Consideración aislada de las cualidades esenciales de un objeto, o del mismo objeto en su pura esencia o noción. Idea o concepto general.
  • Encapsulación Definición: Se refiere al ocultamiento de los datos miembro de un objeto de manera que solo se pueda acceder a través de operaciones definidas para ese objeto.
  • Polimorfismo Definición: Varias formas. Hace referencia a la propiedad por la que es posible enviar mensajes sintácticamente iguales a objetos de tipos distintos. El único requisito que deben cumplir los objetos que se utilizan de manera polimórfica es saber responder al mensaje que se les envía.
  • Herencia Definición: Crear una nueva clase a partir de una clase existente. La nueva clase se denomina subclase y esta contendrá los atributos y métodos de la clase principal o Superclase.
    También existe la herencia múltiple donde se crea una subclase a partir de varias Superclases.
